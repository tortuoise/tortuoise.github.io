Chronicle configuration

In «Manual» by Sridhar

Static site generator written in Perl. Converts text file to html.

Configuration file is /etc/chroniclerc

chronicle --input input_dir/ --output output_dir/ --sitemap-prefix http://www.yourdomain.com/
