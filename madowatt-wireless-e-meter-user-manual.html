Madowatt Wireless E-Meter User Manual

In «Manual» by Sridhar

Madowatt Meter: a current transformer based internet connected meter.

Connecting meter to inverter 1. Clip current transformer onto output of inverter.
2. Connect current transformer to meter via 2.3mm audio jack.
3. Connect 12V DC power supply adapter to meter and mains. Red LED should light up.

Connect XBee adapter to USB port of computer
1. If running Linux can use screen to check readings on serial port.

Download java and madowatt jar 0. Download librxtxSerial.so and save in /usr/lib/jvm/java-7-openjdk-amd64/jre/lib/amd64 (Linux) or in C:\Program Files\Java\jre7\bin (Windoze) 1. Download jar and rxtx and save in classpath.
2. Run jar from command prompt: java -jar Madowatt.jar
3. Application will connect to server and transmit data from serial port.

Monitor Production
1. Connect to Portal
2. Register as new user and select password.
3. Enter meter number as part of registration process.
4. View Production.
